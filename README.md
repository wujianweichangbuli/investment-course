# 投资基础交互课程

这是一个静态网页投资入门课程，当前覆盖 Day 1-14。

## 本地打开

直接打开 `index.html`，或在目录中启动本地服务器：

```bash
python3 -m http.server 8765
```

然后访问 `http://127.0.0.1:8765/`。

## 当前功能

- Day 1-14 正式课程内容
- 测试题、评分和答案解释
- 课程目录折叠
- 当前学习进度
- 概念和西班牙语词汇检索
- 本地浏览器进度保存

## 同步说明

当前版本是纯静态网页。课程内容可以通过 GitHub Pages 同步到网页，但学习进度默认保存在浏览器本地，不会自动跨设备同步。
